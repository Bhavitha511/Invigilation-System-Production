3. SYSTEM SETUP AND INSTALLATION
================================================================================

3.1 ENVIRONMENT PREPARATION
---------------------------
Step 1: Install Python 3.8+
   - Download from python.org
   - Verify installation: python --version
   - Ensure pip is available: pip --version

Step 2: Create Project Directory
   mkdir invigilation-system
   cd invigilation-system

Step 3: Set up Virtual Environment
   python -m venv venv
   
   # Windows
   venv\Scripts\activate
   
   # Linux/Mac
   source venv/bin/activate

Step 4: Install Dependencies
   pip install -r requirements.txt

3.2 DATABASE SETUP
------------------
Step 1: Configure Database Settings
   - Copy .env.example to .env
   - Configure database credentials
   - Set SECRET_KEY and other environment variables

Step 2: Run Database Migrations
   python manage.py makemigrations
   python manage.py migrate

Step 3: Create Superuser Account
   python manage.py createsuperuser
   - Enter username, email, and password
   - This creates the initial administrator account

Step 4: Load Initial Data (Optional)
   python manage.py loaddata initial_data.json

3.3 EMAIL CONFIGURATION
----------------------
Step 1: Configure SMTP Settings in .env
   EMAIL_HOST=smtp.gmail.com
   EMAIL_PORT=587
   EMAIL_USE_TLS=True
   EMAIL_HOST_USER=your-email@gmail.com
   EMAIL_HOST_PASSWORD=your-app-password

Step 2: Test Email Configuration
   python manage.py shell
   >>> from django.core.mail import send_mail
   >>> send_mail('Test', 'Test message', 'from@email.com', ['to@email.com'])
