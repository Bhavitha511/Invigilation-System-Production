WHY THIS EXAM CONFLICTS WITH EVERY FACULTY
===========================================

The auto-allocation algorithm checks timetable clashes using this rule:
  - If a faculty teaches the SAME year as the exam year on that day/time → NO CLASH (class is cancelled, students are in exam)
  - If a faculty teaches a DIFFERENT year on that day/time → CLASH (those students still need teaching)

This exam is for CSE Year 4. So any faculty teaching Year 1, 2, or 3 on Monday 09:00-12:00 will have a CLASH.
Faculty with no Monday slot are still excluded because they belong to ECE/EEE/MECH/CIVIL — same dept exclusion does not apply here,
but the ones below all have direct timetable conflicts.

KEY POINT FOR VIVA
==================
This demonstrates the intelligence of the allocation algorithm:
- It does NOT blindly exclude all faculty with Monday classes
- It checks if the class is for the SAME year (students are in exam → class cancelled → faculty is free)
- Only faculty teaching DIFFERENT year students get flagged as conflicted
- This is the core business logic that makes the system smart
